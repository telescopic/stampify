STAMPIFY

Webpage to STAMP converter


Liked a result? Didn't work? Make sure to leave feedback for a URL here!

Stampify is a tool to automatically convert entertainment/news article web pages to Stories on AMP (STAMP) web pages.

Use the box below to enter the URL you want to STAMPIFY and hit convert. If all goes well, the Stampified page should appear in the iframe.
